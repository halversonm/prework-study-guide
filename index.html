HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user
  • h1 is the largest headline, and h6 is the smallest

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • box-shadow: xpx ypx #____ : creates a box-shadow with x offset and y offset.
  • Make sure to link external style sheet to HTML file you're editing in HTML header.
  • * selector applies the rule to all elements visible on the page.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git pull origin main: Syncs local base branch with remote base branch in Github. Important in team environments in case a team member has added a new feature to the project.
  • git add -A: stage changes. Commits all files in the working branch
  • git commit -m "short, descriptive message of changes implemented": Commits staged changes to our repository. -m flag associates a message with our commit.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • An array is a variable that stores a collection of multiple items.
  • Unlike for loops, functions have to be called in order to execute
